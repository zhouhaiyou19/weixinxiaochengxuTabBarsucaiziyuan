## 使用指南

1. **下载资源**：首先，点击仓库中的下载按钮，将素材文件下载至本地。
2. **选择图标**：根据你的小程序主题和功能模块，挑选合适的TabBar图标。
3. **整合图标**：将下载的图标资源导入到你的小程序项目中，通常放置在项目的`assets`（资产）目录下。
